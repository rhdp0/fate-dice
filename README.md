# Fate Dices

O Fate é um sistema de rpg que surgiu a partir de outro, o FUDGE que usa dados próprios, como um d6, porem este possui: duas faces marcadas com “-“, duas sem marcação e duas com “+“. São usados 4 dados para a rolagem.

Este projeto busca simular estes dados.
